# Needle & Groove

An ultra-premium digital vinyl record-store concept built with **React 18**, **Tailwind CSS**, and **Framer Motion** (all loaded from CDNs, so no install step is required).

## Run locally

Open `index.html` in a modern browser, or serve this directory:

```bash
cd Sol/needle-and-groove
python -m http.server 8080
```

Visit `http://localhost:8080`.

## Interaction

- Hover a sleeve to reveal its vinyl.
- Drag a revealed disc onto the turntable to load it and begin playback.
- Use the circular play/pause control on the turntable to control the animated platter, tonearm, and progress bar.
